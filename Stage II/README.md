This folder is intended to recreate Stage II described in the thesis, the phase in which multiple models are trained and evaluated. At the end of this process, three models are selected based on the following criteria:
Best performance in mite detection;
Best performance in rust detection;
Best performance in cercospora leaf spot detection.
After the selection process, the labels generated by these models are separated so that they can be combined in Stage III and used to train a new model.
The files whose names end with "Detect" must be executed first. These files use the best-performing detection model for each disease and generate new labels, storing them in folders named after the disease that each model detects most accurately.
Next, the files whose names end with "Filter" should be executed. These files filter the detected diseases, retaining only the classes for which each model achieved the highest performance. As a result, new labels are created containing only the specific disease that the previous model detected most accurately, preparing the data for Stage III.
